# Generated Indexes

This directory is generated by `python3 scripts/generate_enriched_catalog.py`.

- `horizontal-index.md`: L1/L2/L3 cross-domain usefulness tiers.
- `type-index.md`: vertical functional categories.
- `dependency-index.md`: API key, tool, runtime, and risk labels.
- `scoring-model.md`: current star-rating policy.
