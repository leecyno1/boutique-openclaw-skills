# Art Direction

This repository uses the local `canvas-design` style workflow as reference:

- define a visual philosophy first (`docs/VISUAL_PHILOSOPHY.md`)
- generate consistent brand assets (`assets/logo.png`, `assets/logo.svg`, `assets/hero.png`, `assets/profiles.png`)

Design language:
- deep navy background
- red signal accent for critical controls
- clean white cards for profile readability

The visual system is intentionally operational, not decorative.
