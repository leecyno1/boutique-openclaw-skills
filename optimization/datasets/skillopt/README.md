# SkillOpt Datasets

Place per-skill optimization datasets here. Each dataset should be small, reviewable, and split into `train/`, `val/`, and `test/` folders.

Recommended layout:

```text
<skill-id>/
  train/items.json
  val/items.json
  test/items.json
```

Keep real credentials, private user data, and proprietary customer documents out of these datasets.
