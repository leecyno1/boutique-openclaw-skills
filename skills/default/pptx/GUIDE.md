# PPTX 使用指南

## 1. 功能定位
- 创建、编辑、重排和导出 PowerPoint。
- 默认档位: 基础档默认安装
- 仓库目录: `skills/default/pptx`
- 安装后目录: `~/.openclaw/skills/pptx`

## 2. 使用前准备
- Python 依赖 `python-pptx`

## 3. 配置步骤
1. 通常无需额外配置。若运行时报缺依赖，再按 `SKILL.md` 补装。

## 4. 推荐提问方式
- 请用 pptx 生成一份 10 页路演稿。
- 请用 pptx 调整这个演示文稿的顺序并替换标题。

## 5. 手动验证
```bash
node -e "console.log("pptx skill ready")"
```
