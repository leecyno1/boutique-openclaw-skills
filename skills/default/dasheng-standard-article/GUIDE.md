# dasheng-standard-article 使用指南

## 1. 功能定位
- 生成并交付“只含完整文章正文”的标准化初稿或终稿（不含口播脚本、视频脚本、标题候选、流程说明）。当用户提到“初稿就是完整文章”“直接覆盖飞书”“按固定标准产出长文（至少5000字、表格、来源）”时使用。适用于 dasheng 选题从 outline 到成稿、质检、飞书覆盖回写与回读验收。
- 默认档位: 仅全量默认包/手动同步
- 仓库目录: `skills/default/dasheng-standard-article`
- 安装后目录: `~/.openclaw/skills/dasheng-standard-article`

## 2. 使用前准备
- 无强制 API Key；按 skill 自身依赖运行。

## 3. 配置步骤
1. 通常无需额外配置。若运行时报缺依赖，再按 `SKILL.md` 补装。

## 4. 推荐提问方式
- 请使用 dasheng-standard-article 帮我处理当前任务。
- 如果 dasheng-standard-article 需要额外配置，请先告诉我缺少什么。
