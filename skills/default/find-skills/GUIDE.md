# Find Skills 使用指南

## 1. 功能定位
- 根据任务需求搜索和推荐适合安装的 skill。
- 默认档位: 基础档默认安装
- 仓库目录: `skills/default/find-skills`
- 安装后目录: `~/.openclaw/skills/find-skills`

## 2. 使用前准备
- 无强制 API Key；按 skill 自身依赖运行。

## 3. 配置步骤
1. 通常无需额外配置。若运行时报缺依赖，再按 `SKILL.md` 补装。

## 4. 推荐提问方式
- 请用 find-skills 找适合做舆情监控的 skill。
- 我想做自动化办公，先用 find-skills 给我列一组 skill。

## 5. 手动验证
```bash
ls -la ~/.openclaw/skills/find-skills
```
