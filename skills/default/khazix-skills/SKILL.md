# Khazix Skills

Khazix Skills is a multi-skill writing and analysis suite from `KKKKhazix/khazix-skills`. Treat this local entry as a suite router rather than a single narrow skill.

Native upstream: https://github.com/KKKKhazix/khazix-skills

## When to Use

Use this skill when the user needs:

- Deep long-form Chinese content or industry report drafting.
- Structured topic analysis and content quality self-checks.
- Writing style templates and logic-density improvements.
- Content ideation, hot-topic analysis, or workflow routing into upstream Khazix subskills.
