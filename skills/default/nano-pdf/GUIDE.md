# Nano PDF 使用指南

## 1. 功能定位
- 自然语言 PDF 编辑与处理。
- 默认档位: 基础档默认安装
- 仓库目录: `skills/default/nano-pdf`
- 安装后目录: `~/.openclaw/skills/nano-pdf`

## 2. 使用前准备
- 视运行环境安装 `nano-pdf` 包

## 3. 配置步骤
1. 先确认 `nano-pdf` 命令或对应 Python 包存在。
2. 若缺失，请按上游发行页安装。

## 4. 推荐提问方式
- 请用 nano-pdf 修改 PDF 中的某段文字。
- 请用 nano-pdf 从 PDF 中抽取表格并整理。

## 5. 手动验证
```bash
ls -la ~/.openclaw/skills/nano-pdf
```
