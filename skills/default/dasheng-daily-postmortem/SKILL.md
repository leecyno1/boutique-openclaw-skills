# dasheng-daily-postmortem

发布后复盘层（L8），收集发布数据，评估评分模型准确性。

## 功能
- 接收发布后的性能数据（浏览量、点赞、分享、评论、互动率）
- 评估成稿价值分的预测准确性
- 评估爆款视频潜力分的预测准确性
- 生成洞察和模型优化建议
- 在飞书创建日期文件夹（YYYY-MM-DD）
- 在文件夹下创建"复盘报告"文档
- 发送复盘完成消息到群

## 触发方式
- 群内发送"复盘"触发
- 手动触发（发布后 24-48 小时）

## 输入
- 已发布内容文件（来自 dasheng-daily-final 输出）
- 性能数据文件（浏览量、点赞、分享等）

## 输出
- 飞书文档：复盘报告（在日期文件夹下）
- 群消息：复盘完成，包含整体评分准确性和文档链接

## 评分准确性评估

### 成稿价值分准确性
- 基于浏览量（60%）和评论数（40%）计算实际分
- 对比预测分，计算准确性百分比

### 爆款视频潜力分准确性
- 基于分享数（50%）和互动率（50%）计算实际分
- 对比预测分，计算准确性百分比

### 综合准确性
- 两个维度准确性的加权平均

## 性能指标
- views: 浏览量
- likes: 点赞数
- shares: 分享数
- comments: 评论数
- engagement_rate: 互动率

## 飞书集成
- 自动创建日期文件夹（格式：YYYY-MM-DD）
- 在文件夹下创建"复盘报告"文档
- 文档包含整体准确性表格、逐条复盘和模型优化建议
