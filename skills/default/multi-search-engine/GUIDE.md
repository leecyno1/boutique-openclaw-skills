# Multi Search Engine 使用指南

## 1. 功能定位
- 17 个搜索引擎聚合，无需 API Key，适合国内外网页搜索与高级操作符检索。
- 默认档位: 基础档默认安装
- 仓库目录: `skills/default/multi-search-engine`
- 安装后目录: `~/.openclaw/skills/multi-search-engine`

## 2. 使用前准备
- 无需 API Key

## 3. 配置步骤
1. 直接按 skill 文档拼接搜索 URL 即可。
2. 更适合用作搜索策略参考，而非单一脚本入口。

## 4. 推荐提问方式
- 请用 multi-search-engine 设计一条跨 Google/百度/Brave 的检索方案。
- 请用 multi-search-engine 给出 site: + 时间过滤的搜索链接。

## 5. 手动验证
```bash
ls -la ~/.openclaw/skills/multi-search-engine
```
