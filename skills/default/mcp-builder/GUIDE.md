# MCP Builder 使用指南

## 1. 功能定位
- 构建 MCP 服务与工具接入规范。
- 默认档位: 基础档默认安装
- 仓库目录: `skills/default/mcp-builder`
- 安装后目录: `~/.openclaw/skills/mcp-builder`

## 2. 使用前准备
- 无强制 API Key；按 skill 自身依赖运行。

## 3. 配置步骤
1. 通常无需额外配置。若运行时报缺依赖，再按 `SKILL.md` 补装。

## 4. 推荐提问方式
- 请用 mcp-builder 设计一个接入企业内部 API 的 MCP server。
- 请用 mcp-builder 审查这个 MCP 服务的工具设计。

## 5. 手动验证
```bash
ls -la ~/.openclaw/skills/mcp-builder
```
