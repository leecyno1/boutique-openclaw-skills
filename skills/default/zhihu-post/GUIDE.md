# zhihu-post 使用指南

## 1. 功能定位
- 在知乎发布内容（专栏文章、想法/动态）。使用 Chrome Browser Relay 控制用户的 Chrome 浏览器完成发布。触发词：知乎发帖、发知乎、知乎文章、知乎想法、zhihu post、发布到知乎、帮我发知乎。
- 默认档位: 仅全量默认包/手动同步
- 仓库目录: `skills/default/zhihu-post`
- 安装后目录: `~/.openclaw/skills/zhihu-post`

## 2. 使用前准备
- 无强制 API Key；按 skill 自身依赖运行。

## 3. 配置步骤
1. 通常无需额外配置。若运行时报缺依赖，再按 `SKILL.md` 补装。

## 4. 推荐提问方式
- 请使用 zhihu-post 帮我处理当前任务。
- 如果 zhihu-post 需要额外配置，请先告诉我缺少什么。

## 5. 手动验证
```bash
ls -la ~/.openclaw/skills/zhihu-post
```
