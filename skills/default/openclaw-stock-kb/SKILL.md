# OpenClaw Stock Knowledge Base

这是一个本地封装的知识库型技能，内容来自仓库内已打包的 `openclaw-stock-kb` 项目资料。

## 适用场景

- 解释某个技术指标、量化策略、情绪分析方法
- 设计股票分析框架、风控流程、回测方案
- 撰写研究笔记、策略手册、投研培训材料
- 给其他股票类技能补充背景知识和方法论

## 本地资料入口

- `strategies/`：量化策略合集
- `indicators/`：技术指标百科
- `sentiment/`：社媒情绪分析方法
- `risk-management/`：风控模型模板
- `tools/`：回测与数据分析工具

## 使用方式

1. 先明确你要找的是策略、指标、情绪方法还是风控模板。
2. 再到对应目录检索相关 markdown 文档。
3. 输出时优先做本地总结，不要原样长篇复制资料。

## 注意事项

- 这是知识库型技能，不是可直接执行交易的自动化脚本。
- 内容仅供研究和学习使用，不构成投资建议。
