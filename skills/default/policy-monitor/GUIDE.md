# policy-monitor 使用指南

## 1. 功能定位
- 监控央行、证监会、发改委、财政部、统计局等官方渠道，自动分类摘要并进行政策影响解读。
- 仓库目录: `skills/default/policy-monitor`
- 原生定位: 本仓库原创封装，官方源列表见 `references/official-sources.md`

## 2. 使用前准备
- 无强制 API Key。
- 需要联网访问官方网页；若环境不能联网，可让用户提供政策链接或网页内容。

## 3. 推荐提问方式
- 请监控今天央行、证监会、发改委有无重要政策更新，并按市场影响排序。
- 请解读这条证监会政策对券商、科技成长和红利资产的影响。
- 请生成本周中国宏观政策周报，只使用官方来源。

## 4. 输出要求
- 每条政策必须保留官方链接、发布日期、机构和分类。
- 明确区分事实、推断和风险，不提供确定性投资建议。
