# Official Policy Sources

## 中国官方渠道

| 机构 | 官网/栏目 | 主要内容 |
|---|---|---|
| 中国人民银行 | https://www.pbc.gov.cn/ | 货币政策、金融市场、支付、汇率、信贷、统计数据 |
| 国家金融监督管理总局 | https://www.nfra.gov.cn/ | 银行保险监管、行政处罚、监管政策 |
| 中国证监会 | https://www.csrc.gov.cn/ | 资本市场监管、发行上市、基金期货、稽查处罚 |
| 国家发展和改革委员会 | https://www.ndrc.gov.cn/ | 宏观调控、产业政策、价格、投资、区域发展 |
| 财政部 | https://www.mof.gov.cn/ | 财政政策、税收、政府债务、预算执行 |
| 国家统计局 | https://www.stats.gov.cn/ | 宏观经济数据、统计公报、月度数据解读 |
| 商务部 | https://www.mofcom.gov.cn/ | 外贸、外资、消费、服务贸易、国际经贸 |
| 工业和信息化部 | https://www.miit.gov.cn/ | 工业、通信、数字经济、专精特新、产业政策 |
| 交易商协会 | https://www.nafmii.org.cn/ | 债券市场、自律规则、处分信息 |
| 上海证券交易所 | https://www.sse.com.cn/ | 交易规则、上市公司监管、市场公告 |
| 深圳证券交易所 | https://www.szse.cn/ | 交易规则、上市公司监管、市场公告 |
| 北京证券交易所 | https://www.bse.cn/ | 北交所政策、上市审核、交易规则 |

|---|---|---|
