# Policy Classification Taxonomy

| 分类 | 关键词 | 常见影响方向 |
|---|---|---|
| 货币政策 | 降准、降息、MLF、LPR、公开市场、流动性、信贷 | 利率、银行、地产、成长股估值、汇率 |
| 资本市场 | IPO、再融资、减持、并购重组、退市、分红、基金、量化交易 | 券商、交易活跃度、风险偏好、主题投资 |
| 产业政策 | 新质生产力、人工智能、半导体、新能源、低空经济、数字经济 | 行业主题、成长板块、供应链 |
| 财政政策 | 专项债、赤字率、税费、补贴、政府采购、预算 | 基建、消费、地方债、周期行业 |
| 金融监管 | 银行、保险、信托、资管、反洗钱、处罚、穿透监管 | 金融机构合规、风险偏好、信用环境 |
| 宏观数据 | GDP、CPI、PPI、PMI、社融、M2、投资、消费、出口 | 大类资产配置、政策预期、行业景气度 |
| 对外经贸 | 关税、出口管制、外资准入、贸易摩擦、制裁 | 出口链、汇率、跨境产业链 |
