# Policy Monitor

面向中文宏观与金融市场的官方政策监控技能。优先使用一手官方来源，不以二手媒体报道作为原始依据。

## 适用场景

- 监控央行、证监会、发改委等官方政策更新
- 生成政策日报、周报、专项政策影响分析
- 按货币政策、资本市场、产业政策、财政政策、宏观数据、监管处罚等分类整理
- 分析政策对 A 股、港股、美股中概、债券、汇率、商品或行业主题的潜在影响

## 核心原则

1. 优先访问 `references/official-sources.md` 中列出的官方渠道。
2. 每条结论必须附官方来源名称、发布日期和链接。
3. 区分“事实摘要”“影响推断”“不确定性”。
4. 不把投资建议写成确定性结论。
5. 如官方页面不可访问，明确说明缺失来源，不用媒体转述替代官方原文。

## 工作流

1. 明确监控范围：机构、日期、政策类型、市场/行业。
2. 检索官方渠道或用户提供的政策链接。
3. 抽取标题、发布日期、发文机构、政策类别、关键条款。
4. 输出结构化摘要和影响分析。
5. 标注需要后续跟踪的配套细则、执行口径或市场反馈。

## 输出格式

```markdown
## 政策摘要
- 标题：
- 机构：
- 日期：
- 来源：
- 分类：货币/资本市场/产业/财政/监管/宏观数据

## 关键内容
1. ...

## 影响解读
- 直接影响：
- 潜在受益/承压方向：
- 不确定性：

## 跟踪事项
- ...
```
