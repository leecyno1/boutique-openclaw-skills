# dasheng-daily-intake 使用指南

## 1. 功能定位
- 大圣自媒体 Intake 最终版（采集阶段唯一入口）：四渠道采集、24小时窗口底层入库、热度归并报告、截图与群回执交付。
- 默认档位: 仅全量默认包/手动同步
- 仓库目录: `skills/default/dasheng-daily-intake`
- 安装后目录: `~/.openclaw/skills/dasheng-daily-intake`

## 2. 使用前准备
- 无强制 API Key；按 skill 自身依赖运行。

## 3. 配置步骤
1. 通常无需额外配置。若运行时报缺依赖，再按 `SKILL.md` 补装。
