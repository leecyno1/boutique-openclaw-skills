# reusable-footage-material 使用指南

## 1. 功能定位
- 聚焦视频素材质量的执行技能。用于单独验证或调优“资料片/纪录片/archive footage 优先、规避口播/访谈/大字幕”的视频检索策略；适合做单点质量测试或素材风格优化，不作为 dasheng 第三环节的默认统一入口。
- 默认档位: 仅全量默认包/手动同步
- 仓库目录: `skills/default/reusable-footage-material`
- 安装后目录: `~/.openclaw/skills/reusable-footage-material`

## 2. 使用前准备
- 无强制 API Key；按 skill 自身依赖运行。

## 3. 配置步骤
1. 通常无需额外配置。若运行时报缺依赖，再按 `SKILL.md` 补装。

## 4. 推荐提问方式
- 请使用 reusable-footage-material 帮我处理当前任务。
- 如果 reusable-footage-material 需要额外配置，请先告诉我缺少什么。
