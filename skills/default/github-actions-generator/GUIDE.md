# github-actions-generator 使用指南

## 1. 功能定位
- 生成 GitHub Actions 工作流，支持 CI/CD、测试、部署等多种场景。
- 默认档位: 仅全量默认包/手动同步
- 仓库目录: `skills/default/github-actions-generator`
- 安装后目录: `~/.openclaw/skills/github-actions-generator`

## 2. 使用前准备
- 无强制 API Key；按 skill 自身依赖运行。

## 3. 配置步骤
1. 通常无需额外配置。若运行时报缺依赖，再按 `SKILL.md` 补装。

- 如果 github-actions-generator 需要额外配置，请先告诉我缺少什么。
