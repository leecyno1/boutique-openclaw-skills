# Web Design 使用指南

## 1. 功能定位
- 做 UI/可用性审查与网页设计建议。
- 默认档位: 仅全量默认包/手动同步
- 仓库目录: `skills/default/web-design`
- 安装后目录: `~/.openclaw/skills/web-design`

## 2. 使用前准备
- 无强制 API Key；按 skill 自身依赖运行。

## 3. 配置步骤
1. 通常无需额外配置。若运行时报缺依赖，再按 `SKILL.md` 补装。

## 4. 推荐提问方式
- 请用 web-design 审查这个页面的可用性问题。
- 请用 web-design 给出一个更适合移动端的设计方向。
