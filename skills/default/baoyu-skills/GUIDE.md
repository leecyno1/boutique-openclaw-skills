# baoyu-skills 使用指南

## 1. 功能定位
- Baoyu 内容产出与分发技能包入口。用于在本地仓库中索引并路由 baoyu 系列子技能。
- 默认档位: 仅全量默认包/手动同步
- 仓库目录: `skills/default/baoyu-skills`
- 安装后目录: `~/.openclaw/skills/baoyu-skills`

## 2. 使用前准备
- 无强制 API Key；按 skill 自身依赖运行。

## 3. 配置步骤
1. 通常无需额外配置。若运行时报缺依赖，再按 `SKILL.md` 补装。

## 4. 推荐提问方式
- 请使用 baoyu-skills 帮我处理当前任务。
- 如果 baoyu-skills 需要额外配置，请先告诉我缺少什么。

## 5. 手动验证
```bash
ls -la ~/.openclaw/skills/baoyu-skills
```
