# 柠檬博士 (Dr.lemon) 文风DNA画像

## 样本概况
- 文章数量：6篇
- 领域：财经投资分析
- 平台：公众号

## 14维风格分析

### 表层特征
1. **词汇选择**：专业但通俗易懂，大量使用投资术语但会解释
2. **句式结构**：长短句结合，喜欢用排比句加强语气
3. **修辞手法**：善用比喻（射箭打靶、精英部队），反问句多
4. **段落长度**：短段落为主，每段1-3句，适合手机阅读

### 结构特征
1. **标题风格**：数字编号+核心观点（如"01"、"02"）
2. **层级组织**：总分总结构，每个部分有明确小标题
3. **信息密度**：高，每段都有核心观点

### 深层特征
1. **论证逻辑**：数据+案例+结论
2. **读者视角**：设身处地，经常用"我们"
3. **态度鲜明**：观点明确，不回避立场
4. **结尾惯例**：操作建议/配置建议

### 独特标记
1. **开头**：场景引入+故事开头（如"昨天跟老王吃饭..."）
2. **数据引用**：大量使用具体数字、年份、百分比
3. **对比手法**：过去vs现在、中美对比、理论与实践
4. **金句**：结尾常有总结性金句

## 标点符号偏好
- ✅ 常用句号、逗号、冒号
- ✅ 常用顿号分层级
- ⚠️ 偶尔使用破折号（知识不能学死了——这是需要注意的）
- ❌ 少用引号（直接引用除外）
- ❌ 不用省略号

## 分块习惯
- 大标题：01/02/03 编号
- 小标题：加粗短句
- 数字列表：1）2）3）或①②③
- 表格：用于对比分析（如沪油/WTI/布伦特对比）

## 段落配方
- 段落长度：1-3句话为主
- 触发一句成段：核心观点、总结、引用
- 过渡段落：常用"而"、"然而"、"所以"

## 叙述方法体系
1. **冲突递进**：先抛出问题→分析原因→给出方案
2. **案例支撑**：身边故事（老王）+ 公开案例
3. **数据说话**：引用研报、数据、图表
4. **类比解释**：用生活化的比喻解释专业概念

## 内容推进方式
1. 从宏观到微观（先讲大趋势→具体到操作）
2. 从问题到方案（先分析痛点→给解决方法）
3. 从过去到未来（历史回顾→未来展望）

## 情绪曲线
- 开篇：场景引入，制造共鸣
- 中段：数据论证，理性分析
- 结尾：给出明确建议，情绪强化

## 起承转合模板

**起**："最近..."、"昨天..."、"XX事件"
**承**："实际上..."、"从数据看..."
**转**："然而..."、"但..."、"不过..."
**合**："所以..."、"结论是..."、"操作建议："
