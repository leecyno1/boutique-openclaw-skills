# Brainstorming 使用指南

## 1. 功能定位
- 在开始做功能、方案或复杂改造前，先做需求澄清与发散。
- 默认档位: 基础档默认安装
- 仓库目录: `skills/default/brainstorming`
- 安装后目录: `~/.openclaw/skills/brainstorming`

## 2. 使用前准备
- 无强制 API Key；按 skill 自身依赖运行。

## 3. 配置步骤
1. 通常无需额外配置。若运行时报缺依赖，再按 `SKILL.md` 补装。

## 4. 推荐提问方式
- 先用 brainstorming 帮我拆解这个需求，再决定怎么做。
- 这个功能先不要写代码，先 brainstorming 出 3 套方案。
