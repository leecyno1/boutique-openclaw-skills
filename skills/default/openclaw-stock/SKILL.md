# OpenClaw Stock Project Adapter

这是一个项目参考型技能，已在本仓库内打包其关键文档，供本地检索和方案复用。

## 适用场景

- 研究自动化交易系统架构
- 对照短线/长线策略和风控设计
- 规划回测、模拟盘和报警流程
- 整理 API Key 配置与免费数据源方案

## 本地资料入口

- `README.md`：整体架构与能力说明
- `QUICKSTART.md`：快速上手
- `API_CONFIGURATION_GUIDE.md`：接口与密钥配置
- `FREE_DATA_SOURCES_GUIDE.md`：免费数据源说明
- `BACKTEST_GUIDE.md`：回测指引
- `AI_TRADING_ADVICE.md`：AI 分析模型接入建议

## 使用方式

1. 优先用它做架构参考、参数模板和部署说明检索。
2. 真正交易前，必须先做回测和模拟盘。
3. 任何实盘流程都应增加人工复核。

## 注意事项

- 这是自动交易项目资料，不应被视为可直接无风险上线的成品。
- 不建议在未完成风控与模拟验证前直接接入真实资金。
