# dasheng-daily-outline

接收选题和素材，生成大纲的环节。

## 功能
- 接收选题和素材包
- 按 Lemon DNA 风格生成大纲
- 大纲结构：Hook → Context → Analysis → Insight → Call to Action
- 在飞书创建日期文件夹（YYYY-MM-DD）
- 在文件夹下创建"大纲"文档
- 发送大纲生成完成消息到群

## 触发方式
- 群内发送"下一步"触发

## 输出
- 飞书文档：大纲（在日期文件夹下）
- 群消息：大纲生成完成，包含文件夹和文档链接

## Lemon DNA 大纲结构

1. **Hook（开篇吸引）** - 用数据或故事快速吸引注意力
2. **Context（背景铺垫）** - 解释为什么这个话题重要
3. **Analysis（深度分析）** - 核心观点和数据支撑
4. **Insight（洞察总结）** - 提炼核心洞察
5. **Call to Action（行动号召）** - 引导读者思考或行动

## 飞书集成
- 自动创建日期文件夹（格式：YYYY-MM-DD）
- 在文件夹下创建"大纲"文档
- 文档包含完整的大纲列表
