# Material Pack Builder

这是一个 **schema / reference** skill，不是默认执行器。

## 用途
- 设计 Material Pack 结构
- 审查素材包字段是否完整
- 补充 evidence gaps / text sources / image sources / video sources 的组织方式
- 为执行型技能提供契约参考

## 默认边界
- 不直接跑 `workflow v2`
- 不直接下载视频/图片
- 不直接更新主流程 manifest

## 默认执行入口
第三环节实际执行请优先使用：`dasheng-daily-material`
