# material-pack-builder 使用指南

## 1. 功能定位
- Material Pack 的 schema / reference 技能。用于设计、审查、优化素材包结构、字段、交付格式与证据缺口清单；适合做方法论和契约分析，不作为 dasheng 第三环节的默认执行入口。
- 默认档位: 仅全量默认包/手动同步
- 仓库目录: `skills/default/material-pack-builder`
- 安装后目录: `~/.openclaw/skills/material-pack-builder`

## 2. 使用前准备
- 无强制 API Key；按 skill 自身依赖运行。

## 3. 配置步骤
1. 通常无需额外配置。若运行时报缺依赖，再按 `SKILL.md` 补装。

## 4. 推荐提问方式
- 请使用 material-pack-builder 帮我处理当前任务。
- 如果 material-pack-builder 需要额外配置，请先告诉我缺少什么。
