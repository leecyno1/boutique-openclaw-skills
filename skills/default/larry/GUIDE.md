# larry 使用指南

## 1. 功能定位
- Marketing growth playbook skill for short-video and social content workflows (TikTok/X/小红书), from idea to publish-ready assets.
- 默认档位: 仅全量默认包/手动同步
- 仓库目录: `skills/default/larry`
- 安装后目录: `~/.openclaw/skills/larry`

## 2. 使用前准备
- 无强制 API Key；按 skill 自身依赖运行。

## 3. 配置步骤
1. 通常无需额外配置。若运行时报缺依赖，再按 `SKILL.md` 补装。

## 4. 推荐提问方式
- 请使用 larry 帮我处理当前任务。
- 如果 larry 需要额外配置，请先告诉我缺少什么。

## 5. 手动验证
```bash
ls -la ~/.openclaw/skills/larry
```
