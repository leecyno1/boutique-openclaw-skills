# dasheng-daily-brief 使用指南

## 1. 功能定位
- 接收聚类结果，生成 Brief 和多维评分的环节。
- 默认档位: 仅全量默认包/手动同步
- 仓库目录: `skills/default/dasheng-daily-brief`
- 安装后目录: `~/.openclaw/skills/dasheng-daily-brief`

## 2. 使用前准备
- 无强制 API Key；按 skill 自身依赖运行。

## 3. 配置步骤
1. 通常无需额外配置。若运行时报缺依赖，再按 `SKILL.md` 补装。
